About Recognition

When streaming, Vinyl AirPlay samples ~20 seconds of audio on an overlapping window (~every 12 seconds) and generates an acoustic fingerprint using Chromaprint — the same engine used by services like Last.fm.

It first checks your local catalog. If the record is already known, it matches instantly with no internet connection. If it's a new record and you have an AcoustID client key, it queries AcoustID (Chromaprint-based fingerprint lookup). Only the fingerprint (not raw audio) is sent. After that, the match + metadata are stored locally, so the lookup never repeats.
